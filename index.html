QDraw

U[n] - moves cursor ↑ n pixels.

D[n] - moves cursor ↓ n pixels.

L[n] - moves cursor ← n pixels.

R[n] - moves cursor → n pixels.

H[n] - moves cursor ↖ n pixels.

E[n] - moves cursor ↗ n pixels.

G[n] - moves cursor ↙ n pixels.

F[n] - moves cursor ↘ n pixels.

[B] - Prefix that moves cursor without drawing.

[N] - Prefix that draws and returns cursor to original position.

M[{+|-}]x%,y% - Moves cursor to point x%,y%. If x% is preceded by + or -, moves relative to the current point.

C[n] - change stroke color 1-4.

A[n] - turns an angle n% degrees.
